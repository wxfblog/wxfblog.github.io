Linux系统下如何配置SSH？如何开启SSH？

2017年4月19日

查看/安装SSH服务

登陆linux系统，打开终端命令。输入 rpm -qa |grep ssh

如果没有安装SSH软件包，可以通过yum 或rpm安装包进行安装

启动SSH服务

安装好了之后，就开启ssh服务。SSH服务一般叫做 SSHD
命令行输入 service sshd start 可以启动

或者使用 /etc/init.d/sshd start

配置\查看SSHD端口

查看或编辑SSH服务配置文件，如 vi /etc/ssh/sshd.config
如果要修改端口，把 port 后面默认的22端口改成别的端口即可

远程连接SSH

如果需要远程连接SSH，需要把22端口在防火墙上开放。
.关闭防火墙，或者设置22端口例外

参看原文：http://jingyan.baidu.com/article/08b6a591f0fafc14a9092275.html
